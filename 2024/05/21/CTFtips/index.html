# 杂项

1.010Editor 是真的强大，没思路可以先看下。

2. 如果压缩包打不开，可以用 010 打开，也可以在 linux 中用 file "文件"，查看文件的原后缀，然后直接改。

3. 一把梭还原图片原本大小，可以在图片打不开时尝试。

4. 图片有时会有隐水注。

5. 一个压缩包是由压缩源文件数据区 + 压缩源文件目录区 + 压缩源文件目录结束标志组成的。用 010 打开一个压缩包，50 4B 03 04，这是压缩源文件数据区头文件标记，50 4B 01 02，这是压缩源文件目录区头文件标记。只需查看 50 4B 03 04 两个字节后面的两个字节 A 和 50 4B 01 02 四个字节后的两个字节 B。

​ （1）A 为 00 00，B 为 00 00，无加密。

​ （2）A 为 00 00，B 为 09 00，伪加密，把 09 改为 00。

​ （3）A 为 00 90，B 为 00 90，真加密。

6.png 文件的开头默认 89 50 4e 47 0d 0a 1a 0a。

7. 摩斯密码是由 “.”“-”“ ” 组成的。

# WEB

1. intval 函数，用于将给定的变量、表达式或字符串转换为整数类型。如果转换的对象是一个浮点数，它会去掉小数部分；如果对象是布尔值或非数字的字符串，它会按照一定规则（比如将真值转换为 1，假值转换为 0）进行转换；对于无法直接转换的字符串，它还可以根据可选的基数（进制）参数来进行解析。

intval () 函数的绕过总结

1）当某个数字被过滤时，可以使用它的 2 进制 / 16 进制来绕过；比如过滤 10，就用 0b1010（八进制）或 0xA（十六进制）。

2）对于弱比较（a==b），可以给 a、b 两个参数传入空数组，使弱比较为 true。

3）当某个数字被过滤时，可以给它增加小数位来绕过；比如过滤 3，就用 3.1。

4）当某个数字被过滤时，可以给它拼接字符串来绕过；比如过滤 3，就用 3ab。（GET 请求的参数会自动拼接单引号）

5）当某个数字被过滤时，可以两次取反来绕过；比如过滤 10，就用～～10。

6）当某个数字被过滤时，可以使用算数运算符绕过；比如过滤 10，就用 5+5 或 2*5

7）当某个数字被过滤时，可以使用 “或” 绕过；比如过滤 10，就用 2 (正确的数) or 10 或 2 || 1000

2.url 编码，如 %2B 代表 +；%20 代表空格；

3. ?flag=rm -rf /* 意在递归且强制删除文件系统根目录下的所有文件和目录，可以导致系统中的所有数据丢失。
